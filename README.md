## FAQ

### ChatGPT Plus 国内为什么经常付款不成功？

常见原因是银行卡地区、账单地址、风控和海外支付环境不匹配。没有可用海外卡时，可以选择微信或支付宝卡密充值。

### 充值时要不要提供密码？

不建议提供密码。更合理的方式是自己登录 ChatGPT，复制 Session，并在确认页核对账号信息。

### Session 复制错了怎么办？

还没提交就重新登录正确账号再复制。已经提交后不要重复下单，保留订单信息联系售后处理。

### 充值后客户端没显示怎么办？

网页端通常更快显示，客户端或移动端可能有同步延迟。可以等待几分钟后刷新，必要时退出重新登录。

### Plus 和 Pro 怎么选？

第一次开通建议先选 Plus，适合多数写作、学习和轻量编程场景。Pro 更适合高频、重度使用者。
